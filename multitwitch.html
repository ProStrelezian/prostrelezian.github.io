GUIDE DU MULTITWITCH

Bienvenue ! Cette page vous permet de suivre simultanément jusqu'à 6 streams différents de manière optimisée.

➕ AJOUTER UN STREAM

Utilisez la barre en haut pour intégrer la chaîne de votre choix (tapez juste le pseudo). Vous pouvez aussi glisser-déposer les fenêtres pour les réorganiser !

📺 PLEIN ÉCRAN & GRILLE

Le mode Plein Écran masque l'interface pour laisser place aux vidéos. La grille s'adaptera automatiquement de manière optimale selon le nombre de streams actifs.

⭐ MODE FOCUS

Cliquez sur l'étoile dans l'en-tête d'un stream pour le mettre en FOCUS : il prendra 80% de l'écran pendant que les autres s'empileront intelligemment sur le côté !

💬 TCHAT INTÉGRÉ

Cliquez sur les onglets pour basculer entre les tchats.
Bug de connexion ? Utilisez le bouton FIX CONNEXION en haut à droite du tchat.

💾 SAUVEGARDE AUTO

Quittez la page et revenez quand vous le souhaitez, vous retrouverez toujours vos streams configurés exactement comme vous les avez laissés !
